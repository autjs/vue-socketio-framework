# vue-socketio-framework
定义 Vue 项目中 Socketio 的使用方案

---

##### 目录

- socketio
  - handler :  协议处理，针对每一个不同的协议需要有一个与之对应的Handler
  - lib : 简单的封装一下socketio
  - agees.js :  协议注册，所有的协议都将在这里出现
  - index.js：初始化相关

---

##### 使用

直接下载 或 clone 到您的项目，在 main.js 里引入 即可，把 .git 直接删除就行（如果有）

```js
import './socketio'
```
